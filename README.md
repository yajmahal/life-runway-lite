# 海外移住 資産寿命シミュレーター（簡易版）

55歳以降に、自分の意思で海外移住生活を始める人向けに、海外生活中の生活費・日本帰国・住まいの売却・公的年金・企業年金・その他収入・運用利回り・インフレ率をもとに、**資産が何歳まで残るか**を日本円ベースで簡易試算する Streamlit アプリです。

- 入力はすべて日本円ベース、金額は「万円」単位
- モンテカルロ（1000 試行固定）を主役にし、参考シナリオを補助的に表示
- 資産がマイナスになる場合は不足額を赤字・アラートで表示
- 結果はルールベースのコメント（総合 / 要因 / 次に試す変更案）で読み解き

> このシミュレーターは、海外移住後の生活設計を考えるための簡易試算です。将来の運用成果、物価、年金、税金、医療費などを保証するものではありません。

> 入力内容はこの試算のためだけに使用し、保存・外部提供・マーケティング利用は行いません。

## ファイル構成

```text
life_runway_lite/
  app.py            # Streamlit アプリ本体（入力UI・結果表示・グラフ）
  simulator.py      # 計算ロジック（資産推移・モンテカルロ・参考シナリオ）
  comments.py       # ルールベースのコメント生成
  requirements.txt
  README.md
```

計算ロジック（`simulator.py`）とコメント生成（`comments.py`）は UI から分離しています。

## セットアップと起動方法

### 1. 仮想環境を作成して有効化

```bash
cd life_runway_lite
python3 -m venv .venv
source .venv/bin/activate        # Windows の場合: .venv\Scripts\activate
```

### 2. 依存パッケージをインストール

```bash
pip install -r requirements.txt
```

### 3. アプリを起動

```bash
streamlit run app.py
```

ブラウザが自動で開きます。開かない場合はターミナルに表示される `http://localhost:8501` を開いてください。

## 使い方

1. 入力条件（海外生活を始める年齢、手元資産、生活費、年金など）を設定します。
2. 画面下部の **「計算する」** ボタンを押すと試算が実行されます。
   - スライダーを動かすたびに再計算はされません。条件を決めてからボタンを押す方式です。
3. 結果として「資産が残る可能性」「標準 / 厳しめ / 良好ケース」「資産が尽きる目安」「資産推移グラフ」「参考シナリオ」「コメント」が表示されます。

## 試算の前提（簡易版）

- 計算期間：海外生活を始める年齢〜100歳
- 初期資産：手元資産 ＋（売却する場合）日本の持ち家の手取り額
- 生活費：インフレ率を毎年反映
- 収入：公的年金・企業年金・その他収入を開始年齢以降に年額で加算（終身は100歳まで）
- 公的年金：受給開始年齢に応じた簡易係数（60歳=76.0% 〜 70歳=142.0%）で補正
- 運用：年初資産に利回りを反映 → そのあと収入・支出を加減（コード内で一貫）
- モンテカルロ：各年の利回り＝想定利回り＋正規分布の変動（標準偏差 8%、上下限でクリップ）。乱数シード固定で同じ入力なら同じ結果

本簡易版には、為替レート・現地通貨入力・健康保険・税金や社会保険の詳細計算・医療費/介護費の詳細・一時支出・保存機能・PDF 出力・AI コメントなどは含みません。
